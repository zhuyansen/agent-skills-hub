Privacy Policy

Last updated: April 2, 2026

Information We Collect

Agent Skills Hub collects minimal data:

  • Email address — Only if you subscribe to our weekly newsletter. Used solely to send you the newsletter.
  • Analytics — We use Plausible Analytics, a privacy-friendly, cookie-free analytics tool. No personal data is collected, no cookies are set, and all data is aggregated.

How We Store Your Data

Newsletter subscriber emails are stored in Supabase (PostgreSQL) with row-level security enabled. Only verified, active subscribers receive emails. You can unsubscribe at any time via the link in every newsletter.

No Cookies

This website does not use cookies. Plausible Analytics is fully cookie-free and compliant with GDPR, CCPA, and PECR without requiring a cookie banner.

Data Sharing

We do not sell, trade, or otherwise transfer your personal information to third parties. We do not run ads. Repository data displayed on this site is sourced from the public GitHub API.

Third-Party Services

  • GitHub API — For sourcing public repository data.
  • Supabase — For data storage and subscriber management.
  • Plausible Analytics — For privacy-friendly traffic analytics.
  • Resend — For sending newsletter emails.
  • GitHub Pages — For hosting the website.

Contact

For privacy-related inquiries, please open an issue on our GitHub repository.
