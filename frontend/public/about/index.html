About Agent Skills Hub

The largest open-source directory of AI agent tools. Helping developers discover the right tools to build intelligent agents.

Our Mission

Agent Skills Hub exists to solve a simple problem: the AI agent ecosystem is growing so fast that developers struggle to find the right tools. We continuously index GitHub repositories, score them for quality, and organize them into actionable categories so you can find exactly what you need in seconds.

How It Works

  1. Discover — Our sync engine searches the GitHub API every 8 hours for new and updated AI agent repositories.
  2. Analyze — Each tool is scored on 10 weighted signals (stars, commit activity, documentation quality, license, and more) to produce a 0–100 composite score.
  3. Classify — Tools are automatically categorized into 53 use-case scenarios using keyword matching and topic analysis.
  4. Serve — The directory is served as a fast static site via GitHub Pages, with Supabase PostgreSQL as the data layer.

Open Source

Agent Skills Hub is fully open source. The entire codebase — frontend, backend, sync engine, and scoring algorithms — is available on GitHub. Contributions, bug reports, and feature requests are welcome.
