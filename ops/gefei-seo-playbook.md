## 1. AI 时代 SEO 变天(印证我们的 GSC 发现)
**哥飞**:AI Overview + 广告 + YouTube + PAA 挤压自然结果,"排名第一点击率也极低"。
→ 我们 GSC 实测 pos 6-9 却 0.1-0.4% CTR = 同一现实,不是 bug。
⚠️ 措辞校准:这不是纯 AI 时代新现象——**SERP 功能块(精选摘要→AI Overview)长期持续挤压高位 CTR**(Brainly 案例:精选摘要时代排第一也只拿 ~25% 搜索量)。收益形态从"点击"迁移到"被引用",水位要同时盯 GSC clicks 和 AI 引荐/被提及数。
**解药(=我们该做的)**:
- **寄生虫 SEO**:把安全报告/数据集发去 Reddit/HN/LinkedIn/Medium 高权重站,借权重(接 arXiv/HF 弹药)。**选站升级**:①Ahrefs most-cited-domains 系列(Copilot/Grok/Gemini 各自最常引用的域名榜)= 按目标 AI 引擎选靶,不再凭感觉;②**反向工程 AI grounding**:用户视角问 ChatGPT 买家问题("recommend a tool to check if a Claude skill is safe"),展开 thinking 看它搜了什么词、去哪些站搜 → 检索词进 /best/ 选题池,引用域名进外链清单。**HN 细则**:Show HN 格式 `Show HN: [名] – 短描述`,单次提交≈10+ 镜像站外链;我们的钩子是数据结论("扫了 13 万 skill,X% 有红旗")而非产品。
- **底部漏斗竞品拦截页**:"LobeHub alternative"、"is X safe"(/audit/ 已是雏形,扩)。
- 场景长尾:/best/{scenario} 继续挖具体词。
- **视频对冲**:YouTube 视频可直接进 SERP 参与排名,且视频提及是 AI 可见性最强信号(0.737)——博客/报告视频化是第三条对冲渠道。

## 2. 品牌词:哥飞一句话定案
**哥飞**:"谷歌算法更新永远利好品牌,域名选好记的品牌名,别用 bestaixxx.com。"
→ 印证"Agent Skills Hub 类目词吃亏",但"更新利好品牌"=把品牌权威做起来 Google 越站你 → **权重线(arXiv/PR/数据)是对的,坚持**。

## 3. 新词挖掘:我们有现成雷达
**哥飞**:Google Trends 非源头;新词先在游戏主播/模型公司推特/AI 玩法博主(宝玉归藏)火;倒推源头+订阅 RSS+Twitter API 定时抓(半年内/高赞/带链接)。
→ 我们 sync 每 8h 扫全网 skill = 本赛道新词雷达;/skill/+/audit/ 天生第一时间落地。
**补的动作**:每周跑 `fetch_gsc.py brand` + compare,盯 rising queries,新工具名一被搜就确保排第一。**rising 新工具词建页 SLA:24h**(新词生命线以小时计,官宣→上站→首批外链压 3-7 天)。

**Trends 分工注(终结表述矛盾)**:Trends 做**发现**滞后 3-15 天(非源头成立);做**验证已知词的季节性/趋势方向 + AI 扩词根**(新版"建议搜索字词")好用。两个场景别混。轻量替代自建管道:grok.com 原生 Task 定时任务,每日 X 新词上奏折,零工程量。

**雷达新通道(三条,均领先 Trends)**:
- **query.domains 域名抢注溯源**:新词爆发前域名先被抢注;溯到一手信源(HF/arXiv/@_akhaliq 类账号)长期盯。
- **外链平台新提交流**:新站上线必发外链,盯 Toolify/PH/聚合站的新收录 = 别人帮你盯新词。
- **社媒热度源**:X/Reddit 讨论领先搜索 1-2 周(传播路径:圈内讨论→关键节点转发→出圈→主动搜寻),正是抢首发收录的窗口。

**老词对偶策略(存量供需比打法)**:新词雷达之外的另一条腿——竞争者每年自然消亡(域名到期/弃站),谷歌需要新供给。判型 = 趋势平稳 1-5 年 + `allintitle:` 供需比 <0.25 + SERP 前十多论坛/视频帖(供给不足)。/best/ 选题不必只追 rising,老工具词的劣质在位者也是机会。**长尾反向吃主词**:同一需求的冷门表达先拿下,借语义匹配+权重积累反吃主词;判老词可做性:SERP 前两页有独立站首页(非仅大站内页)+ 这些站外链域数百级 = 有机会。

**选题侦察四步法(Canva 前增长总监,60 天 +226%)**:①竞对社媒高转发内容逐篇精读;②Pinterest 搜领域热词看大众偏好;③Quora/Reddit 按"求答人数+最高赞"记录问题库(选题只写已验证有需求的);④Ahrefs 反查竞对博客外链来源,滚动维护"未来会分享我们内容的伙伴"库。→ /blog/ 选题从"我们做了什么"转向"Reddit/Quora 在问什么"。

## 4. 外链:两免费招+一避坑(印证 16 行框架)
**哥飞**:
- **Bing 站长 backlinks 工具** = 免费看竞品外链,只显示必应已收录(自带质量过滤)。
- nofollow 来自高权重(HN)照样传权重,别嫌弃。
- Guest Post 买 $100+;DR 虚高低流量模板站不买。
→ 印证外链 16 行(#7/#15 别碰垃圾高 DR)。**新动作**:用 Bing backlinks 扒 LobeHub/smithery 外链发布地照着铺(备选交叉:SEMrush/SimilarWeb 反链导出,覆盖 Bing 没爬到的新站;快速抽查用 ahrefs 免费 backlink-checker)。

**补充细则(2026-07-23 大师库对照)**:
- **内链先于外链(哥飞四句话)**:先让首页/频道页给目标页做内链,内链权重不够再对外找外链。
- **锚文本分级**:`<a href title="关键词">关键词锚文本</a>`(最好)> 有锚文本无 title > 裸域名/URL(最差)。主动求链/投稿时显式要求带关键词锚文本。
- **好外链 8 项清单**(投放前 10 秒过一遍):有真实流量(搜索词含品牌词)· 近期有收录(`site:` + 最近一周)· 有 DR · 有运营时长 · 相关性高 · dofollow 优先 · **页面出链少** · **尽量发达国家**。工具:Ahrefs Toolbar 常显 DR + AITDK 看流量/域龄。
- **节奏纪律**:新站单月外链突增 100+ 会被判异常;自然增长曲线优于爆发(我们 07-10 单日 12+ 开口属多元非付费型,风险低,但 8 月对账要看 Ahrefs 引用域曲线自查)。
- **付费外链双门槛**:①对方站有真实流量且内容相关(风险门槛);②我们台账验收通过(ROI 门槛)。只满足其一都不买。**采购渠道(niko 会议 07-23)**:Adsy 平台自采客座博客,绕开服务商差价(行情参考:3 条 ≈ ¥1600 属正常价);付费 guest post 节奏**每日 1-2 条封顶**(与"单月突增 100+ 判异常"同一条纪律的日颗粒度版);收录核验三件套 = AITDK + Ahrefs Toolbar + GSC 外链报表。
- **免费外链定位拍板(07-23)**:陈荣涛"免费外链收录率<10%、导航站流量≈0"——对通用站成立,对我们**部分成立**:免费外链/导航站降级为"**发现层**"(收录触发 + AI grounding 语料),不再当权重层扩量;已投的 17 站潮只验收不追加。例外:垂类精准导航(libhunt 类程序员目录)+ awesome 列表仍值得,那是真实垂类流量不是权重游戏。
- **GitHub 辟谣与正确用法(07-23)**:"300 star 换高权 dofollow 外链"是**不实传言**——GitHub 的价值只有一个:**垂类用户导流载体**。正确姿势 = 联系开源仓库作者谈 README 广告位/资源置换(程序员群体引流效率极高,2-3 个稳定仓库即覆盖初始流量)——我们的候选:Agent Reach、cc switch 类 agent 生态仓库,天然同受众。极端下行案例:Genius 2013 年因有偿求链被整站 K(连品牌词都搜不到)——"操纵性购买"不是低 ROI 问题,是生存问题。
- **新渠道**:赞助高 DR 开源项目换赞助栏外链($5-5000,开发者工具站契合;⚠️灰色地带,标注风险)· 见执行手册 ROI 表新增的免费高 DR 四件套。
- **反面情报**:过期域名 301 权重迁移是灰帽主流(9000 站案例)——不做,但验收外链/评估竞品 DR 时警惕过期域名注水;这也解释了 Bing"优质引用域"口径为何比 Ahrefs 严。

## 7. 后 5 条会议新捞干货
- ⚠️ **`.top` 域名被点名"不受搜索引擎信任、难收录"**(记录③)。我们就是 .top。叠加"类目描述名" = 品牌权重上不去的两刀。**短期不迁**(11万页+外链+GSC历史重来风险高),靠 arXiv/HF/PR 堆品牌权威;品牌真起来后再考虑 .com/.ai。
- **全页面用"高搜索量表述"选词**:"Midjourney prompts">>"Midjourney inspiration";登录/定价页都能优化。→ 用 GSC 数据校准 /category//best/ 标题精确措辞(MCP server vs servers、Claude skills vs Claude Code skills)。
- **截流四类词:需求词/竞品品牌词/模型词/typo 词**。模型词转化最高——**但有边界条件**:仅适用已建立使用场景心智的成熟生态词(Claude Code/Cursor);炒作期裸模型名(WAN2.2/GLM4.5 刚发布头几天)搜索者只想知"是什么",无付费意愿,等意图从"是什么"转向"怎么用"再上承接页。→ 建模型名落地页 + 竞品 alternative 页。**typo 词**:查 GSC 是否已有 "cluade skills / agent skils" 类变体曝光,有则现有页 FAQ/别名吸收(不建独立页防薄页)。
- **找大佬抄盘**:大词短时上首页/新词最快冲上去=大佬;抄其外链+新关键词;`intitle:` 看结果增速判断词会不会爆(**注意 intitle=或、allintitle=且;评估竞争度/算 KGR 必须用 allintitle**)。→ 盯 LobeHub/smithery/glama。**抄盘工具三件套**:①Similarweb 关键词研究→着陆页→筛"新点击量" = 竞品新上且起量的页面;②Semrush 主要页面按自然流量排序;③**竞品 sitemap 每日 diff 监控**(新增 URL = 其新建页/新词,直接进建页候选)。**AI 流量也能抄盘**:SimilarWeb AI 流量面板输竞品域名,看哪些落地页吃 AI 流量、什么 prompt 触发。**广告位也能抄盘**:Ads Transparency Center / sitedata 反查竞品投放词——肯花钱买的词 = 验证过的商业意图词。
- **定价页 3 档锚定中间 + 默认年费**(年费总价<月费年化)。→ $49/enterprise 定价页照此。
- **冷门细分 > 热门模型词**(排名易度):模型词截流拿意图,长尾拿排名,两条腿 = 印证我们 /best//audit/ 策略。
- 小语种:子目录竞争小反哺整站权重,但**关键词不能直译要查本地真实搜索词**。中文 SEO 若认真推再用,优先级低。

## 7b. Reddit 执行细则(从一句话扩成 SOP,2026-07-23 补)

> 原状态:寄生虫 SEO 里一句"发去 Reddit"。三个信源(增长心流 7 案例研究 / niko 常青流量池 SOP / 毅行评论卡位)交叉后的完整版:

1. **判型先行**:目标 subreddit 是"禁自我营销"(借干货/工具间接展示)还是"允许"(可直发)。发帖前读版规。
2. **Karma 先攒**:新号先在大社区/新帖下发建设性评论攒基础分,否则直接被判 spam 删帖;需要时 mod 私信话术(老用户身份+价值说明)换代发/豁免。
3. **前期价值(Front-loaded Value)**:先发干货帖攒声誉(Trennd 创始人 57 个趋势帖),火了再顺势带产品——不是直接发链接。
4. **常青流量池**:`site:reddit.com <关键词>` 找排名靠前+活跃+监管松的旧帖,评论区/编辑插入;Google 给 Reddit 帖权重高,一条优质评论可吃数月到数年;追踪表记关键词/帖链/浏览量。
5. **风控**:帖内不直贴链接,发帖几天后再编辑加;跨社区用原生 crosspost 不重复手发;冷 DM ≤15/天。
6. **评论卡位口诀**:"小爆蹲30,大爆抢10"(中V帖等赞数 30-50 再评,大V帖赞数 <10 就抢);评论三模板:感同身受/交流探讨/顺势引流,忌空洞夸赞。
7. **弹药**:100 个开发者向 subreddit 清单见 niko 库(按 SaaS/创始人/营销/技术产品等 7 类);最佳发帖窗 = 北京时间上午 10-12(北美东部晚 9-11)。
8. **投放判断预防针**:"主题匹配 ≠ 购买意图匹配"——r/ClaudeAI 的人会点"is this skill safe"原生帖(兴趣),未必是愿为审计付费的人(意图)。

## 7c. 混合套利站识别(审计/评级差异化弹药,2026-07-23 补)

拆竞品/给 skill 评级时,查 **sitemap 结构 + 子域 + AdSense publisher ID** 可识别"双引擎结构"站:体面 SaaS 门面拿信任,真正吃量的是需求词发动机(多语言下载页/内容子域挂广告)。案例:greenconvert(AI 转写门面 + YouTube 下载器词发动机)、quickutils(工具壳 + 5 个内容子域)。这类站的"表面工具、实际流量套利"判型,是我们信任层评级的差异化维度。

## 8. 数据四件套交叉验证法(2026-07-06 实战)
**没有哪个工具单独够用 —— 交叉验证才抓得出单工具看不见的问题。**

四件套分工:
- **GSC**(`ops/gsc/fetch_gsc.py`,代理 7897)= 搜索曝光/点击/词。找主题、rising、品牌词。
- **Plausible**(`ops/plausible/fetch_plausible.py`)= 真实 PV/来源/热门页。⚠️07-04 才建,历史短。
- **Clarity**(`ops/clarity/fetch_clarity.py`,**必须直连**,10次/日)= 死点击/怒点/JS错。找 UX 摩擦。
- **GA**(analytics-mcp,查 **property 485523739 + hostName** 过滤,别查空的 534610783)= 完整站内行为/转化。历史最全。

**实战抓到的三类问题(单工具都看不见):**
1. **重复页 keyword cannibalization**:GSC 视角只看词,看不出"我新建的 ppt-skills 和现有 ppt-presentation 抢同一批词"。**Plausible 的真实 PV 列表**才暴露现有页存在 → 合并到有历史的 slug、删新页。**教训:建场景页前先 `grep slug` 查现有 + 看 Plausible 热门页,别只信 GSC。**
2. **GA 属性错位**:站点 gtag 数据流进「新媒体运营」(485523739),专属属性(534610783)是空的。**Plausible 数据少(07-04才建)会误导你以为流量小,GA 才是真规模(2837会话)**。交叉对比两个分析工具才发现。
3. **死点击定位**:Clarity by-url 的 `Url` 字段(不是 url)+ `subTotal` 才是每页死点击数。热点=看着像按钮的 `<div>` 卡片/纯 `<img>` 头像。修法:能点的给链接,没干净目标的别硬加。

**流程**:GSC 找主题 → **Plausible 查是否已有页(防重复)** → 目录查品类广度 → 建/合并 → GA 看转化 → Clarity 修 UX 摩擦。

## 工具站打法(2026-07-12 · 哥飞《为什么你的工具站有页面却拿不到好排名?》· 目录站适配)

> 原文讲工具站,我们是目录站 —— 只搬能映射的,标清哪条不适用。

### ✅ 能搬(3 条)
1. **"工具功能 + 图文 + 结果 = 一个页面"**:页面信号别拆散。映射到我们:skill/audit 页把**答案(安全评级)+ 动作(安装命令)+ 证据(红旗清单)** 合一个 URL。audit 页已做对("Is X safe?" 首屏给结论)。
2. **聚合页 > 单结果页**:围绕有搜索量的关键词做聚合页,别把每个结果做成薄页。我们的 /best/ 场景页、/category/ 页就是聚合页;数据驱动建页(ppt/obsidian)= 正解。
3. **新站先验证少数页出词再扩产**:与水系冷启动铁律互印证(pos 8-20 阈值要 GSC 数据,先播种后修剪)。

### ⚠️ 敲警钟(1 条,直指我们软肋)
> "新站权重不够时,大量薄/高度相似页面,搜索引擎爬完发现没几个能排名,会降低整站可见度。"

**DR 6 + 133K skill 页 + 7,924 audit 页** = 正是这个风险场景。虽然我们的页是真数据(非垃圾生成),但"模板高度相似 + 低权重"这一刀真实。**对冲已在做**:排水闸(仅索引 stars≥50)、内链通管、noindex 会员页。结论:**这篇是再次确认排水纪律的重要性,不是要推翻什么** —— 但记分板里的模板超长(3,666 标题 / 3,368 描述)+ 孤儿页(1,871)得盯着,别让"薄页面群"拖低整站。

### 📋 实测:/best/ 场景页第一屏能行动吗?(哥飞最可执行的一条)
- **结论:通过(目录站版)**。首屏顺序 = H1 → "🔍 浏览 N 个工具" 数据条 + 榜首推荐 → "Top N 工具"卡片列表(可点进详情装)。**首屏领着的是排好序的工具列表,不是一屏介绍文字**。
- **唯一差距**:无页内实时筛选框。但静态页(static-first 架构)下,排好序的列表本身就是交付物 —— 用户扫一眼点进去即可,不该为加个 filter 破坏预渲染。**不改**。
- 若将来要更进一步:把榜首那张卡做得更突出 / 加个"跳到全站搜索"链接,属锦上添花,非紧急。

### ❌ 不适用
"工具页和落地页合并" —— 那是单功能工具站的问题。目录站天然"搜索→列表→详情"三层,这是对的信息架构,不合并。
