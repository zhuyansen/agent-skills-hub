# 收款与跨境资金运维手册(2026-07-23 · 源自赫兹/吴就业/niko 实战库)

> 定位:五件套之外的独立运维档。SEO 手册 C2/C2b 管"$49 收银台怎么防翻车",本档管"整个收款-提现-合规链路"。
> 信源:赫兹 25 篇支付系列 + niko Stripe 封号复盘 + 吴就业收款周记。均为一手实操,非教科书。

## 一、渠道冗余红线(最重要的一条)

**有用户持续付款的站,必须 ≥2 个收款方案。** 实例:PayPal 收第一笔款即被封(常见现象)、Creem 有被整体封禁冻结的先例、Stripe 违规连坐。单渠道被封 = 收入归零。
降级序:Stripe → PayPal/Creem 备用链接 → mailto(仅分钟级过渡)。

## 二、Stripe

### 接入与配置
- 香港个人注册:dashboard.stripe.com/register 选香港;+86 手机;护照代身份证;银行填众安"户口号码"(非卡号);身份文件审核阶段需香港节点;审核期间即可收款;收款一周后可提现(2 天到账)。
- 结算货币 = 收款货币(USD):设置→商家→银行账号与货币加美元设默认;否则每单 2% 兑换费(美元结算提现 1%)。
- 账单描述符 = 网站名(品牌一致 + 防争议第一道)。
- 优惠券:`discounts`(自动应用)与 `allow_promotion_codes`(用户手填)**互斥,同设报错**;exit-intent 折扣用 coupon + 第二条 Payment Link,不动主链接。
- 年费引导:产品加年费价格 → 月费价格详情勾"向上销售",Checkout 原生出月/年切换,零前端开发。

### 风控与争议
- Radar 风控团队版 + 3DS 规则(请求验证/阻止/审查)+ 风险评分>50 阻止;盗刷特征 = 付了不用、多卡尝试。
- 争议 SOP:**永不点"同意"**(照扣 $15-20,按地区浮动以 Dashboard 为准)→ 先邮件让用户撤销争议再退款(截图即证据)→ 证据按时间线整理不贴链接 → "自动提交证据"手动开启;可疑订单主动退款。争议率过高 = 封号。
- **封号连坐机制(niko 实录)**:一个子账户违规 → 同主体全部子账户审查;~3 次后新建子账户直接被封;再犯封主体身份,最严封个人 ID 匹配即秒封。申诉:下架违规内容 → 逐个提交审查说明 → 打回换内容再申诉 → 单独开 case 跟踪。NSFW/蹭牌等灰色内容提前清理。
- 多站复用:新建**账号**(风险隔离)> 同账号新建产品(一站出事全家连坐);左上角账号→创建单独账户,商家信息复用。

### 数据
- Stripe API 免费拉全量订单(金额/币种/国家/卡种/退款/争议)自建看板,替代付费 Sigma;utm 编进 client_reference_id 做订单级归因。
- Stripe CLI:`stripe login` 后让 AI 按模板口喷建产品+webhook;线上须 `sk_live_` 密钥。

## 三、PayPal
- 个人卖家:paypal.cn/portal/account-selection;身份证+人脸;商业名称填**通用品牌名**(多站复用);职业"工程技术人员"、分类"计算机软件开发"、销售额 0-1万刀档;约 20 分钟过审。
- 接入:developer.paypal.com JS SDK,让 AI 复刻现有 Stripe 支付路径;沙盒用 Sandbox personal 账号测。实测有用户主动选 PayPal = 双渠道有真实增量。
- ⚠️ 收第一笔即被封是常态:电话客服/哥飞 PayPal 群申诉;解封后 **60 天无争议才能提现**(实测可能 ~20 天,勿依赖),现金流要预留。
- 提现到港卡:选"美国或中国香港"银行;**众安填法:分行代码=账号前 3 位,账号=剩余位数**;实际到账汇率比 Google 低 2-3%(隐性损耗计入定价);3-5 天到账。
- 政策:禁平台代第三方商家统一收款(将来若做"代 skill 作者收认证费"绕开 PayPal)。

## 四、Creem 与三方政策对表
- Creem 提现:直打支付宝会被拒(支付宝只认同名转入,Creem 是公司主体打款)→ **用自己的 Wise 账户中转**(打款人=本人);Wise 风控严,先小额测。
- 三方共同禁区:AI 成人/赌博/传销。Stripe 额外:交友、加密/NFT、众筹、远程医疗、打赏平台需预审。Creem 对 AI 产品最细:禁换脸/Deepfake、论文代写、第三方下载器、IPTV、PLR 转售;**套壳命名规则**:产品名不能直接用模型名("VEO3Studio"✗,"MintAI—powered by VEO3"✓);生成类强制接 Moderation API;营销词禁 uncensored / no filter / NSFW / 18+。
- 这份对表同时是 /audit/ 的评级素材(skill 商业化合规维度)。

## 五、跨境账户链路
- **Wise 个人**:手机验证→姓名全大写注意 First/Last 顺序→结算币种→中国税务居民填身份证号→扫码人脸。用途:Creem 中转、多渠道收款汇集。
- **香港银行卡(线下)**:港澳通行证签注 + **提前开国际漫游收验证码**(没开导致跑三趟的实案)+ 出入境记录(12367)线上预约;现场免费 WiFi 仅 30 分钟断网重来;牢记自设用户名密码。
- **Mercury 水星(需美国公司+EIN)**:邀请注册各得 $250(90 天内卡内 1 万美金);地址证明要**带公章**银行账单(众安无章被拒,招行带章过);收入证明 = Stripe payments/payouts + 银行到账 + 账户信息四截图拼 PDF;多轮打回属正常。
- **美国公司**:globalgo 比 Stripe Atlas 便宜(Wyoming LLC,2 天办结;Atlas 有支付被拒先例)。
- **美元回流消费**:众安卡绑 WeChat Pay HK(年 10 万港币免手续费额度);Stripe 美元结算→众安内部换汇,满 $1000 有汇率券。

## 六、定价与手续费
- Stripe 2.9%+$0.30:$4.99 有效费率近 10%、$9 约 7%、$19/$29 回落 ~5% —— 低价档按最坏手续费倒推底价。
- CAC 反推定价(niko):欧美 CPC $1+,按 15% 付费率,100 点击 ≈ $145 成本,单付费 ≥$9 才回本 —— 财务定价法与哥飞"3 档锚定"心理定价法互补。
- 敢于定价:AppSumo 3800 产品数据,$60-80 档总收入 = $0-20 档 76 倍;高价存活率更高。
