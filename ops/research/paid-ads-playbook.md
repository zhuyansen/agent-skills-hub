# 付费投流手册(暂缓执行 · 2026-07-23 · 源自赫兹 15+ 篇 + niko + 七鹿投放实录)

> **当前定位:不投。** 对我们唯一说得通的用途是"付费意愿验证器"(给 Pro 漏斗喂受控样本),不是获客渠道。
> 与避坑 15"渠道打透一个"的关系:投流按"一次性产品验证实验"归类(封顶预算 + 单一假设 + 到期必关),不占渠道心智。

## 〇、投流 Gate 三条(一条不齐,一分钱不投)

1. **GA 关键事件已标**(⚠️ 这是现在就卡着的:MEMORY 里"GA 标 key event"用户待办未完成——它同时是投流的硬前置)。
2. 自助付费闭环**至少有过自然成交**(audit_pro_upsell 样本在等)。
3. 单客毛利(LTV − API 成本)≥ 目标 CPA。
第 0 条:决定投的那天,**先领新客券再建 campaign**——投 $500 送 $500(另有 $1000/$1500 档),必须未登录状态从活动入口注册新账号领取,账号一旦有消耗数据永久失去资格。

## 一、Google Ads 基础机制
- **质量分三要素**(预估 CTR / 广告相关性 / 落地页体验)决定排名与实付 CPC,出价高 ≠ 排前——和自然位的 message-match 是同一原理。
- CPC 竞价:设最高出价,实付常低于;日预算单日最多上浮 2 倍,月度按均值找平。
- Keyword Planner:免费看全市场搜索量区间(GSC 之外的词库盲区补充);投放账号有消耗后区间变精确值。
- 结算国家一旦选定不可改;无外国护照/永居一律选中国(否则 AdSense 验证过不了)。

## 二、投放纪律(实战验证)
- 目标选"销售/潜在客户",不选"网站流量";竞价先"点击次数"积累数据,**攒够 ~5 个不同购买用户**再切"转化次数"/智能出价(过早换挡 = 算法重新学习断单)。
- 只投搜索网络(官方亲口:展示网络转化率低);排除低 GDP/无支付网关地区;避免广泛匹配;持续否词(negative keywords)。
- **不动盈利中的 campaign**:要测新出价策略就复制 campaign 单独跑对照,改原配置会打断已学习状态(实测断单案例)。
- 预算跟效果走不平均分;campaign 别贪多(管不过来 = 预算分散)。
- 选词:具体词 > 宽词("text to video ai generator" 转化好,"ai video" 烧钱);看竞品广告反复用的词 = 市场验证过的词。
- 频次上限防骚扰;新受众先"观察"模式攒数据再正式定向。
- 学习路径:support.google.com/google-ads 官方文档,不拼凑零散教程。

## 三、Reddit Ads 实录(目录站预防针)
完整参数包($42.57 实验):开户新账号+美区 USD 即可;绑卡 $10 预授权;日限 $5 实跑 $6+;15,302 展示 / 165 点击 / CPC $0.26 / CTR 1.078% / **1 人到付款页 / 0 成交**。
- 核心教训:**"目标人群主题匹配 ≠ 购买意图匹配"**(r/UnsentLetters 自我疗愈型 vs r/breakups 情绪宣泄型,产品动作与用户真实诉求有"心理距离")。映射到我们:r/ClaudeAI 的人会点 "is this skill safe" 原生帖(兴趣),未必愿为审计付费(意图)。
- 素材:模仿社区原生帖(CTR 1.633%)> 标准 SaaS 广告。
- Reddit Pixel 回传站内行为 = 归因/再营销基建(我们现有 GA4 是单向的)。
- 若测:预算封顶 $50,当一次付费意愿实验,别当渠道。
- 平台后台点击数 vs GA 会话数必有口径差,以站内关键事件为准;UTM 必打。

## 四、判定与止损
- 小额测试 ROI 速判:$1000 回 $900 可继续;回 $300 = 产品/落地页问题,不是投放问题(niko 阈值)。
- "点击是兴趣,付款才是意图"——验收链路:支出 → 会话 → 关键事件 → 付费,一层都不能跳。
- **"投流反哺 SEO"是未证实假说**(信源用词全是"好像/感觉",n=1),不作为立项理由。
- 广告位变现负面清单:目录站挂 Monetag/Adsterra 类低质广告网络 = 砍"信任层"人设;affiliate 若做必须显式披露且不影响评级排序(评级收佣金 = 人设塌方)。
