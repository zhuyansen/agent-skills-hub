# 渠道匹配决策树(2026-07-23 · 源自 niko 渠道思考 + 七鹿垂类观察 + 良辰美模式分类)

> 五件套此前最大的空白:只讲"怎么把 SEO/GEO 做好",没讲"**什么产品/需求该用什么渠道**"。本档是建站/上新品前的分诊步骤。

## 一、渠道四问(niko 框架)

对任何新产品/新页面,先答四问:
1. **用户会不会主动搜?** 不搜 → SEO 无水可接,走社媒/红人。
2. **能否几秒演示秒懂?** 能 → 短视频/演示 GIF 是强渠道;不能 → 内容/文档型渠道。
3. **是否依赖情绪/信任?** 是 → 红人背书/社区口碑先行,广告和 SEO 都慢。
4. **能否被自然说成一个搜索词?** 能 → SEO 主线;不能 → 先造词(品牌词)再承接。

## 二、认知阶段匹配器

| 用户认知阶段 | 心理 | 吃这段流量的渠道 |
|---|---|---|
| 不知道问题存在 | "这是什么?" | 社媒红人/短视频(制造问题意识) |
| 知道问题,不知道方案 | "怎么办?" | 社媒 + 信息型 SEO(指南/科普) |
| 知道方案,在比工具 | "哪个好?" | **Google Search 主场**(对比页/best 页/alternative 页) |
| 已选定,在用 | "怎么用好?" | 文档/教程 SEO + Email |
| 老用户 | 留存/复购 | Email / 产品内 / 常驻工具 |

关键推论:**模型词/新词的转化率取决于词所处的认知阶段**——炒作期裸模型名停在第一阶段(无付费意愿),成熟生态词已到第三阶段(转化最高)。判断新词该不该建承接页,先判它的阶段。

## 三、垂类分诊(何时 SEO 优先,何时不是)

- **低搜索量/低 CPC 垂类**(多数 AI 工具站):启动期社媒/红人快起量,SEO 当放大器——"新词新站打法不适合 AI 工具,别抱流量爆火心态"(七鹿)。
- **高搜索密度垂类**(工具词/游戏词/参考查询):SEO 可硬拼进水口。
- **强互动产品**(游戏/每日机制):体验和留存信号可部分抵消权威劣势;**参考型/目录型站**(我们)不适用此路,严守关键词纪律。
- **红海同质化品类**:免费流量窗口趋零,"SEO 本质是买流量",要么付费杠杆要么别进——我们靠结构性差异化(数据/信任层)不属此类。

## 四、起量模式五分类(良辰美,选题/对标时判型用)

1. **事件型**:蹭突发事件,爆快跌快(355万→77万);必配回流机制(订阅/提醒/可嵌组件)才留得住。
2. **品牌词承接型**:承接已有品牌搜索(anime-sama 2025万/月);能用不能照抄——蹭词侵权是死路。
3. **工具词型**:稳定需求词 + 工具交付(quickutils);我们主模式。
4. **fandom 圈层型**:小圈子反复打开(rpedia 7万);对应我们 = 特定 agent 生态子社区。
5. **母体导流型**:短视频/社媒母体喂站(rotgen 评论关键词→自动私信带 ref);对应我们 = X 日报喂 /daily/。

判"别人站为什么起量"四指标:峰值留存率 · 注册→达标间隔 · 直接访问占比(>50%=品牌记忆)· Top keywords 是否品牌词。

## 五、我们的映射(结论)

- 主线不变:SEO/GEO 双进水口 + X 打透(第三认知阶段 + 工具词型是我们的主场)。
- 补的短板:①第一/二阶段的内容(短视频/信息图)只做资产复用不做主渠道;②事件承接页(水系 ⑥ 已补);③Email 段(newsletter 已有,做深"评级变化告警"回访钩子)。
- 不做:红人投放、付费获客(见 paid-ads-playbook Gate)、母体导流的全自动化(风控)。
